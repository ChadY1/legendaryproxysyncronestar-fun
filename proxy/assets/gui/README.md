# GUI textures StarLife RP

Placez ici les PNG fournis (extraits du ticket) pour que les GUIs SQript les utilisent :

- `fuel_pump.png` : écran de pompe à essence (quantité + bouton Acheter).
- `atm_withdraw.png` : grille de retrait rapide (10/20/50/100/500/1000 €).
- `atm_home.png` : menu principal ATM (Withdraw / Deposit / Transaction Log / Close).
- `atm_denied.png` : bannière d'échec (ATM Bank Denied).
- `atm_success.png` : bannière de succès.
- `atm_hacking.png` : barre de progression de piratage (HACKING ATM BANK AUTOMATIC).
- `atm_hacked.png` : écran de piratage réussi.
- `vehicle_grid.png` : interface d'achat de véhicule (Acheter).
- `starlife_brand.png` : logo StarLife RP (utilisable dans les boutons).

Les noms doivent correspondre aux références du script `proxy/sqript/gui_star_life.sq`.
